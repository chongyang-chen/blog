### 字符的输入输出

```
int putchar(int ch);
int getchar();

```

### 字符串输入输出

```
int puts(char *str);
char *gets(char *str);
```

### 格式输出函数

```
printf(格式控制，输出列表)
```
格式字符：d， i， o, x, X, u, c, s, f, e, E, g, G;

###格式输入函数

```
scanf(格式控制，地址列表)
```
