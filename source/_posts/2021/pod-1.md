pod install 与 pod update 的区别。

<!-- more -->

### install

```sh
pod install    Install project dependencies according to versions from a
		Podfile.lock
```

### update

```shell
pod update     Update outdated project dependencies and create new
     Podfile.lock
```

这个是使用pod 命令时给出的解释。下面这篇文章给出了很好的解释。
